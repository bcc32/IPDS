1. Four basic elements of counter-controlled repetition:
    1. Counter variable
    2. Initialization of counter
    3. Loop repetition condition (e.g., counter <= x)
    4. Incrementation of counter
  2. The while statement repeats a statement as long as a condition is true, but the for statement includes two other statements: an initialization and an increment statement (commonly used for counter-controlled repetition). Both statements repeat a statement as long as a condition holds, but the for statement has "added features".
  3. It would be more appropriate to use a do...while statement rather than a while statement if, say, the programmer wants the user to keep trying until they correctly input a value; the do-while loop would always execute at least once, and this saves two lines of code (prompt and accept).
  4. Both the break and continue statments cause the program to skip the rest of the loop body, but the break statement causes the program to exit the loop as well; the continue statement does not.
